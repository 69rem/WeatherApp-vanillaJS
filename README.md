# Weather App

## Overview
A simple web application to display current weather information for any city. The app fetches data from the OpenWeatherMap API and shows temperature, weather description, and other details.

## Features
- Search weather by city name
- Display temperature, weather description, and more
- Responsive and interactive design
- Data fetched from OpenWeatherMap API

## Deployment
The app is deployed via GitHub Pages:
[Live Demo](https://69rem.github.io/WeatherApp-vanillaJS/)

## Usage
1. Enter the name of a city in the search field.
2. Click "Search" or press Enter.
3. View the current weather information for the city.
